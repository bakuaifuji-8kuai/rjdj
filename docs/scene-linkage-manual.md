# 场景联动页面功能操作手册

## 一、模块概述

### 1.1 模块定位
场景联动模块是智能场景模块的重要组成部分，主要用于创建、管理和控制自动化规则。该模块通过设定触发条件和执行动作，实现系统的智能化自动控制，降低人工干预频率，提高隧道照明系统的运行效率和安全性。

### 1.2 适用场景
- 人员检测联动：检测到人员时自动开启相应区域照明
- 光照度联动：根据环境光照度自动调节亮度
- 安全报警联动：检测到漏电等安全隐患时自动断电
- 定时控制：在指定时间段内执行特定规则
- 应急响应：紧急情况下自动触发应急场景

## 二、页面功能说明

### 2.1 场景联动主页面

#### 页面布局
页面采用左右布局，左侧为系统导航菜单，右侧为场景联动主区域。

#### 功能区域划分

| 区域 | 功能说明 | 位置 |
| :--- | :--- | :--- |
| **视图切换** | 在卡片模式/列表模式之间切换 | 页面顶部左侧 |
| **新增按钮** | 跳转至联动管理页面创建新规则 | 页面顶部右侧 |
| **批量删除按钮** | 批量删除选中的规则（需选中规则后可用） | 页面顶部右侧 |
| **搜索筛选区** | 按规则名称、启用状态、优先级筛选规则 | 搜索栏 |
| **规则展示区** | 展示所有规则，支持勾选、启用/禁用 | 页面主体 |
| **分页控件** | 分页浏览规则列表 | 页面底部 |

#### 视图模式说明

**卡片模式**：
- 5列网格布局展示规则卡片
- 直观展示优先级、启用状态、执行日期
- 适合快速浏览和状态管理

**列表模式**：
- 表格形式展示规则信息
- 支持单选/多选操作
- 提供编辑和删除操作按钮
- 适合详细信息查看和批量操作

#### 规则卡片/列表项元素说明

| 元素 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| **选择框** | 勾选规则，支持批量操作 | 可同时选中多条规则 |
| **优先级标签** | 显示规则优先级（高/低） | 高优先级为红色，低为绿色 |
| **规则名称** | 显示规则标识和名称 | 便于识别和搜索 |
| **启用开关** | 控制规则的启用/禁用状态 | 蓝色为启用，灰色为禁用 |
| **执行日期** | 显示规则的生效时间范围 | 支持一直执行或时间区间 |
| **编辑按钮** | 进入联动管理页面编辑规则 | 列表模式独有 |
| **删除按钮** | 删除单条规则（需确认） | 列表模式独有 |

#### 搜索筛选功能

| 筛选条件 | 说明 | 选项 |
| :--- | :--- | :--- |
| **规则名称** | 按名称关键词搜索 | 支持模糊匹配 |
| **启用状态** | 按启用/禁用状态筛选 | 全部、启用、禁用 |
| **优先级** | 按优先级筛选 | 全部、高、低 |

### 2.2 联动管理页面（新增/编辑）

#### 页面布局
页面分为基本信息区和配置区，配置区包含触发条件和执行动作两个标签页。

#### 基本信息配置

| 字段 | 说明 | 操作要求 |
| :--- | :--- | :--- |
| **规则名称** | 规则的唯一标识 | 必填，支持中文/英文/数字 |
| **优先级** | 规则的执行优先级 | 必填，高/低二选一 |
| **执行日期** | 规则的生效时间 | 必填，一直执行/定时执行二选一 |

#### 触发条件配置

| 字段 | 说明 | 选项示例 |
| :--- | :--- | :--- |
| **产品类型** | 触发设备的类型 | 灯控制器、传感器、摄像头 |
| **设备** | 具体的触发设备 | 展厅隧道灯、光亚展灯杆 |
| **触发类型** | 触发的方式 | 事件触发、属性触发 |
| **触发条件** | 具体的触发阈值或事件 | 检测有人、检测无人、光照度低、漏电 |

> 支持添加多个触发条件，多个条件为"或"关系

#### 执行动作配置

| 字段 | 说明 | 选项示例 |
| :--- | :--- | :--- |
| **产品类型** | 执行设备的类型 | 灯控制器、集控、传感器 |
| **设备** | 具体的执行设备 | 展厅隧道灯、光亚展灯杆 |
| **动作内容** | 具体执行的操作 | 开灯、关灯、调光50%、断电 |

> 支持添加多个执行动作，多个动作按顺序执行

#### 触发时间段配置

可设置规则的生效时间段，仅在指定时间内触发规则。

## 三、操作步骤

### 3.1 新增联动规则

**功能入口**：场景联动页面 → 「新增」按钮

**操作步骤**：
1. 在左侧导航菜单点击「智能场景」→「场景联动」进入页面
2. 点击页面顶部的「新增」按钮
3. 系统跳转到联动管理页面
4. 填写基本信息：
   - 输入规则名称（必填）
   - 选择优先级（高/低）
   - 选择执行日期类型：一直执行或定时执行
5. 配置触发条件：
   - 在「触发条件」标签页中
   - 点击「添加触发条件」按钮添加条件
   - 选择产品类型、设备、触发类型、触发条件
   - 可重复添加多个触发条件
   - 点击×按钮删除不需要的条件（至少保留1条）
6. 配置执行动作：
   - 切换到「执行动作」标签页
   - 点击「添加执行动作」按钮添加动作
   - 选择产品类型、设备、动作内容
   - 可重复添加多个执行动作
   - 点击×按钮删除不需要的动作（至少保留1条）
7. （可选）设置触发时间段：
   - 点击「触发时间段」按钮
   - 在弹窗中选择开始时间和结束时间
   - 点击「确定」保存
8. 点击页面顶部的「保存」按钮完成创建
9. 点击「取消」按钮放弃操作

**流程图**：
```
点击新增 → 填写基本信息 → 配置触发条件 → 配置执行动作 → （可选）设置时间段 → 保存
```

### 3.2 编辑联动规则

**功能入口**：场景联动页面 → 列表模式 → 目标规则 → 「编辑」按钮

**操作步骤**：
1. 在场景联动页面切换到「列表模式」
2. 找到目标规则
3. 点击该规则的「编辑」按钮
4. 系统跳转到联动管理页面，自动填充该规则的信息
5. 修改需要调整的内容：
   - 基本信息（规则名称、优先级、执行日期）
   - 触发条件（可添加/删除/修改）
   - 执行动作（可添加/删除/修改）
   - 触发时间段
6. 点击「保存」按钮保存修改
7. 点击「取消」按钮放弃修改

### 3.3 启用/禁用规则

**功能入口**：场景联动页面 → 规则卡片/列表 → 启用开关

**操作步骤**：
1. 在场景联动页面找到目标规则
2. 点击规则上的「启用/禁用」开关
3. 系统弹出提示确认变更
4. 开关变为蓝色表示已启用，灰色表示已禁用

### 3.4 搜索和筛选规则

**功能入口**：场景联动页面 → 搜索筛选区

**操作步骤**：
1. 根据需要设置筛选条件：
   - 输入规则名称关键词
   - 选择启用状态（全部/启用/禁用）
   - 选择优先级（全部/高/低）
2. 点击「搜索」按钮执行筛选
3. 页面显示匹配的规则
4. 点击「重置」按钮清空所有筛选条件，显示全部规则

### 3.5 删除单条规则

**功能入口**：场景联动页面 → 列表模式 → 目标规则 → 「删除」按钮

**操作步骤**：
1. 在场景联动页面切换到「列表模式」
2. 找到目标规则
3. 点击该规则的「删除」按钮
4. 在弹出的确认对话框中点击「确定」
5. 规则被成功删除
6. 点击「取消」放弃删除操作

### 3.6 批量删除规则

**功能入口**：场景联动页面 → 勾选多条规则 → 「批量删除」按钮

**操作步骤**：
1. 在场景联动页面勾选需要删除的规则（可在任意视图模式下勾选）
2. 确认「批量删除」按钮变为可用状态
3. 点击「批量删除」按钮
4. 在弹出的确认对话框中点击「确定」
5. 选中的规则被成功删除
6. 点击「取消」放弃删除操作

### 3.7 切换视图模式

**功能入口**：场景联动页面 → 「卡片模式」/「列表模式」标签

**操作步骤**：
1. 点击「卡片模式」标签切换到卡片视图
2. 点击「列表模式」标签切换到列表视图
3. 系统保留当前的筛选条件和分页信息

### 3.8 分页浏览

**功能入口**：场景联动页面 → 底部分页控件

**操作步骤**：
1. 点击「上一页」/「下一页」按钮翻页
2. 直接输入页码跳转到指定页
3. 查看页面总数和总记录数

## 四、注意事项

### 4.1 优先级原则
- 高优先级规则先于低优先级规则执行
- 同一优先级规则按创建顺序执行
- 冲突规则以高优先级为准

### 4.2 规则启用/禁用
- 禁用的规则不会被触发，即使条件满足
- 禁用状态不影响规则的编辑和查看
- 紧急情况下可快速禁用无关规则

### 4.3 删除确认机制
- 所有删除操作（单条/批量）均需二次确认
- 确认对话框显示要删除的规则数量，便于核对
- 删除操作不可恢复，请谨慎操作

### 4.4 触发条件逻辑
- 多个触发条件为"或"关系，满足任一条件即可触发
- 事件触发：检测到特定事件时立即触发
- 属性触发：设备属性满足特定条件时持续触发

### 4.5 执行动作逻辑
- 多个执行动作按添加顺序依次执行
- 前一个动作完成后才会执行下一个动作
- 单个动作失败不影响后续动作的执行

### 4.6 触发时间段
- 未设置时间段时，规则在整个生效期内都可触发
- 设置时间段后，仅在指定时间内触发
- 时间段为24小时制，精确到分钟

### 4.7 数据校验
- 规则名称为必填项，不能为空
- 至少需要1个触发条件和1个执行动作
- 优先级和执行日期为必选项

### 4.8 页面权限说明
- 该页面需要管理员权限访问
- 创建、编辑、删除规则需要相应的操作权限

## 五、常见问题

### Q1：规则创建后为什么没有自动触发？
**A**：请检查以下几点：
1. 规则是否处于启用状态
2. 触发条件是否满足
3. 是否在触发时间段内（如设置）
4. 是否在执行日期范围内

### Q2：如何调整规则的执行顺序？
**A**：通过调整优先级实现：高优先级规则先执行；同一优先级按创建顺序执行。

### Q3：批量删除时不小心选错了规则怎么办？
**A**：在确认对话框中点击「取消」即可放弃删除操作。确认删除后无法恢复，请谨慎操作。

### Q4：可以设置多个触发条件同时满足才触发吗？
**A**：当前版本多个触发条件为"或"关系，满足任一条件即可触发。如需"与"关系，请联系系统管理员配置。

### Q5：规则执行失败如何排查？
**A**：请检查：
1. 执行设备是否在线
2. 网络连接是否正常
3. 设备权限是否足够
4. 动作内容是否有效

### Q6：触发时间段可以设置多个吗？
**A**：当前版本仅支持设置一个连续的时间段。如需多个不连续的时间段，请创建多条规则。
