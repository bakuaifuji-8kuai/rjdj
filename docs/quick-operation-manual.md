# 快捷操作页面功能操作手册

## 一、模块概述

### 1.1 模块定位
快捷操作模块是智能场景模块的核心功能之一，主要用于对预设场景进行快速执行和管理。该模块提供直观的场景卡片展示方式，支持场景搜索、执行、状态查看等功能，帮助用户高效地控制隧道照明系统中的各类场景。

### 1.2 适用场景
- 紧急情况下快速执行应急场景（如应急开灯）
- 日常运维中快速切换不同照明模式
- 查看场景下发历史记录，追踪执行状态

---

## 二、页面功能说明

### 2.1 页面布局
页面采用左右布局，左侧为系统导航菜单，右侧为快捷操作主区域。

### 2.2 功能区域划分

| 区域 | 功能说明 | 位置 |
| :--- | :--- | :--- |
| **标签页切换** | 切换场景控制/场景下发记录视图 | 页面顶部左侧 |
| **新增按钮** | 跳转至场景管理页面创建新场景 | 页面顶部右侧 |
| **搜索区域** | 按场景名称搜索过滤场景 | 场景控制页内 |
| **场景卡片网格** | 展示所有场景，支持勾选、显示控制、执行操作 | 场景控制页主体 |
| **场景下发记录表** | 展示场景执行历史记录 | 场景下发记录页 |

### 2.3 场景卡片元素说明

每个场景卡片包含以下元素：

| 元素 | 功能说明 | 操作方式 |
| :--- | :--- | :--- |
| **选择框** | 勾选场景，支持批量操作 | 点击复选框 |
| **场景名称** | 显示场景标识和名称 | 展示信息 |
| **显示开关** | 控制场景在卡片网格中的显示/隐藏状态 | 点击开关切换 |
| **执行按钮** | 执行该场景（需二次确认） | 点击按钮 |
| **执行结果按钮** | 查看该场景最近一次执行结果 | 点击按钮 |
| **更多操作** | 展开更多操作选项 | 点击三点图标 |

### 2.4 场景下发记录字段说明

| 字段 | 说明 | 示例 |
| :--- | :--- | :--- |
| **No** | 记录序号 | 1 |
| **场景名称** | 执行的场景名称 | 应急开灯 |
| **下发时间** | 执行时间戳 | 2024-04-22 10:26:58 |
| **状态** | 执行结果状态 | 成功/失败 |
| **详情** | 执行结果描述 | 执行成功，共控制156盏灯具 |

## 三、操作步骤

### 3.1 执行场景

**功能入口**：场景控制页面 → 目标场景卡片 → 「执行」按钮

**操作步骤**：
1. 在左侧导航菜单点击「智能场景」→「快捷操作」进入页面
2. 确认当前处于「场景控制」标签页
3. 在场景卡片网格中找到目标场景
4. 点击场景卡片上的「执行」按钮
5. 在弹出的确认对话框中点击「确定」按钮
6. 系统弹出「执行结果」对话框，显示执行详情
7. 点击对话框关闭按钮完成操作

**流程图**：
```
进入页面 → 选择场景 → 点击执行 → 确认执行 → 查看结果 → 关闭对话框
```

### 3.2 查看执行结果

**功能入口**：场景控制页面 → 目标场景卡片 → 「执行结果」按钮

**操作步骤**：
1. 在场景控制页面找到目标场景卡片
2. 点击「执行结果」按钮
3. 在弹出的对话框中查看执行时间、状态和详情信息
4. 点击对话框关闭按钮

### 3.3 搜索场景

**功能入口**：场景控制页面 → 搜索输入框

**操作步骤**：
1. 在搜索输入框中输入场景名称关键词
2. 点击「查询」按钮或按回车键进行搜索
3. 页面显示匹配的场景卡片
4. 如需重置搜索条件，点击「重置」按钮

### 3.4 切换场景显示状态

**功能入口**：场景控制页面 → 场景卡片 → 显示开关

**操作步骤**：
1. 在场景卡片上找到「显示/隐藏」开关
2. 点击开关切换状态：蓝色为显示，灰色为隐藏
3. 隐藏的场景将不再在卡片网格中显示

### 3.5 新增场景

**功能入口**：页面顶部右侧 → 「新增」按钮

**操作步骤**：
1. 点击页面顶部的「新增」按钮
2. 系统跳转到场景管理页面
3. 在场景管理页面填写场景信息：
   - 场景名称（必填）
   - 显示状态（显示/隐藏）
   - 排序序号（数字）
   - 动作配置（可选，可添加多条）
4. 点击「保存」按钮完成创建
5. 点击「取消」按钮放弃操作

### 3.6 查看场景下发记录

**功能入口**：页面顶部左侧 → 「场景下发记录」标签

**操作步骤**：
1. 点击「场景下发记录」标签页
2. 页面展示所有场景执行历史记录表格
3. 可通过滚动查看更多记录

## 四、注意事项

### 4.1 执行确认机制
- 所有场景执行操作均需二次确认，防止误操作
- 确认对话框显示要执行的场景名称，便于用户核对

### 4.2 场景显示控制
- 隐藏的场景仅不在卡片网格中显示，仍可通过搜索找到
- 显示状态不影响场景的可执行性

### 4.3 执行结果有效期
- 「执行结果」按钮显示的是该场景最近一次的执行结果
- 执行结果包括成功和失败两种状态

### 4.4 批量操作支持
- 场景卡片支持勾选选择
- 勾选后的场景可进行批量操作（需配合其他功能模块）

### 4.5 搜索功能说明
- 搜索支持模糊匹配，不区分大小写
- 搜索范围仅限于场景名称字段

### 4.6 页面权限说明
- 该页面需要管理员权限访问
- 执行场景需要相应的操作权限

## 五、常见问题

### Q1：执行场景时提示权限不足？
**A**：请联系系统管理员确认您的操作权限。

### Q2：为什么有些场景卡片显示为灰色？
**A**：灰色表示该场景处于隐藏状态，点击显示开关即可恢复显示。

### Q3：执行结果显示失败怎么办？
**A**：请查看详情信息，常见原因包括设备离线、通信故障等，可联系运维人员排查。

### Q4：如何创建自定义场景？
**A**：点击页面顶部「新增」按钮，进入场景管理页面进行创建。
