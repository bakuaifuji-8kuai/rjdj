
# 隧道照明回路控制功能操作手册

## 一、功能概述

隧道照明回路控制功能是城市隧道智慧照明系统的核心模块，用于对隧道内各照明回路进行集中监控和远程控制。系统支持回路状态实时监测、亮度调节、故障告警等功能，确保隧道照明的安全、节能和智能化管理。

## 二、页面结构

### 2.1 页面入口

进入系统后，点击左侧菜单栏 **隧道管理 > 隧道照明** 即可进入本页面。

### 2.2 功能标签页

页面包含两个功能标签页：

| 标签页 | 功能描述 |
| :--- | :--- |
| **调光控制** | 管理隧道各分段的照明调光配置 |
| **回路控制** | 管理各照明回路的状态和亮度控制 |

## 三、回路控制功能操作

### 3.1 回路列表查看

1. 进入隧道照明页面后，点击 **回路控制** 标签页
2. 系统显示所有照明回路的列表，包含以下信息：

| 列名 | 说明 |
| :--- | :--- |
| 选择框 | 用于批量选择回路 |
| 回路编号 | 回路的唯一标识（如 HL-001） |
| 回路名称 | 回路的名称（如照明回路1、应急回路） |
| 状态 | 回路运行状态（正常/故障） |
| 负载率 | 当前负载百分比 |
| 功率 | 回路功率（kW） |
| 关联设备 | 该回路控制的灯具数量 |
| 操作 | 控制、删除按钮 |

### 3.2 回路筛选

1. 在回路控制页面的筛选栏中，点击 **选择回路** 下拉框
2. 选择需要筛选的回路（全部回路、回路1、回路2）
3. 点击 **查询** 按钮，列表将显示筛选结果

### 3.3 回路调光控制

#### 操作步骤：

1. 在回路列表中找到需要控制的回路
2. 点击该回路操作列的 **控制** 按钮
3. 弹出「回路调光」对话框，显示以下内容：
   - **回路名称**：当前选中回路的名称（只读）
   - **亮度(%)**：滑块和数字输入框，范围 0-100%
   - **预设按钮**：快速设置亮度为 0%、25%、50%、75%、100%

4. 调节亮度：
   - **方式一**：拖动滑块到目标亮度值
   - **方式二**：直接在数字输入框中输入亮度百分比
   - **方式三**：点击预设按钮快速设置

5. 点击 **确定** 按钮，系统将下发调光指令
6. 操作成功后显示提示信息：`[回路名称] 调光成功，亮度: [亮度值]%`

#### 示例：

```
操作场景：将"照明回路1"亮度调整为75%

1. 找到回路编号为 HL-001 的"照明回路1"
2. 点击操作列的"控制"按钮
3. 在弹出的对话框中，点击"75%"预设按钮
4. 点击"确定"按钮
5. 系统提示："照明回路1 调光成功，亮度: 75%"
```

### 3.4 回路删除

1. 在回路列表中找到需要删除的回路
2. 点击该回路操作列的 **删除** 按钮（红色文字）
3. 系统执行删除操作（注：实际系统中通常会有确认对话框）

### 3.5 批量删除回路

1. 在回路列表中勾选需要删除的多个回路
2. 点击页面顶部的 **批量删除** 按钮（红色按钮）
3. 系统批量删除选中的回路

## 四、调光控制功能操作

### 4.1 调光配置列表

1. 点击 **调光控制** 标签页
2. 系统显示隧道各分段的调光配置列表，包含以下信息：

| 列名 | 说明 |
| :--- | :--- |
| 选择框 | 用于批量选择配置项 |
| No | 序号 |
| 分段-照明类型 | 隧道分段类型（入口段、基本段、出口段） |
| 凌晨 | 凌晨时段调光值 |
| 白天 | 白天时段调光值 |
| 傍晚 | 傍晚时段调光值 |
| 夜间 | 夜间时段调光值 |
| 操作 | 修改、删除按钮 |

### 4.2 筛选条件

1. **展厅隧道/梅溪湖隧道**：选择隧道名称
2. **展厅隧道洞**：选择隧道洞名称
3. 点击 **查询** 按钮应用筛选条件

### 4.3 新增调光配置

#### 操作步骤：

1. 点击页面顶部的 **新增** 按钮
2. 弹出「修改」对话框，填写以下配置项：

| 配置项 | 说明 | 必填 |
| :--- | :--- | :--- |
| 分段类型 | 入口段/基本段/出口段 | 是 |
| 分段名称 | 具体分段名称 | 否 |
| 照明级别 | 基本照明/加强照明/应急照明 | 是 |
| 执行模式 | 光感/时序/手动 | 是 |
| 传感器类型 | 光照度传感器/CO传感器 | 是 |
| 启用 | 是否启用该配置 | 否 |
| 照度值1 | 照度范围下限（lx） | 否 |
| 照度值2 | 照度范围上限（lx） | 否 |
| 调光值 | 调光百分比（0-100%） | 否 |
| 是否启用 | 开关控制 | 否 |

3. 填写完成后点击 **确定** 按钮保存配置
4. 系统提示：`调光配置保存成功`

### 4.4 修改调光配置

1. 在调光配置列表中找到需要修改的配置项
2. 点击操作列的 **修改** 按钮
3. 在弹出的对话框中修改配置参数
4. 点击 **确定** 按钮保存修改

### 4.5 删除调光配置

1. 在调光配置列表中找到需要删除的配置项
2. 点击操作列的 **删除** 按钮（红色文字）
3. 系统执行删除操作

## 五、状态说明

### 5.1 回路状态

| 状态 | 标识颜色 | 说明 |
| :--- | :--- | :--- |
| 正常 | 绿色 | 回路运行正常 |
| 故障 | 红色 | 回路存在故障，需检修 |

### 5.2 配置状态

| 状态 | 说明 |
| :--- | :--- |
| 已配置 | 该分段已设置调光参数 |
| 未配置 | 该分段尚未配置调光参数 |

## 六、注意事项

### 6.1 操作权限

- 回路控制和调光配置操作需要管理员权限
- 普通用户仅能查看，无法进行修改操作

### 6.2 安全提示

1. **应急回路**：应急回路用于紧急情况照明，请勿随意调整或关闭
2. **故障回路**：状态为"故障"的回路需要及时检修，避免影响隧道照明安全
3. **负载率监控**：当回路负载率超过80%时，建议及时检查设备状态

### 6.3 节能建议

1. 根据时段自动调整亮度（凌晨、白天、傍晚、夜间）
2. 使用光感模式，根据外界光照强度自动调节
3. 非高峰期可适当降低照明亮度

### 6.4 故障处理

1. 发现故障回路时，及时通知维护人员
2. 记录故障时间和故障现象
3. 在维修完成前，可启用应急照明作为备用

## 七、常见问题

### 7.1 调光操作无响应

**可能原因**：
- 回路处于故障状态
- 网络连接异常
- 权限不足

**解决方案**：
- 检查回路状态，故障回路需先修复
- 检查网络连接
- 联系管理员确认权限

### 7.2 亮度调整后未生效

**可能原因**：
- 执行模式设置为"时序"，需等待时序触发
- 设备离线
- 指令下发失败

**解决方案**：
- 检查执行模式设置
- 检查设备在线状态
- 重新执行调光操作

### 7.3 负载率过高

**可能原因**：
- 连接设备过多
- 设备故障导致电流异常
- 线路老化

**解决方案**：
- 检查并减少关联设备
- 检测设备运行状态
- 安排线路检修
